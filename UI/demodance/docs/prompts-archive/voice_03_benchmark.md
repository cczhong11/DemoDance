# VOICE Benchmark

## 0.1 Clarity (0-5)
- Sentences concise
- No ambiguity
- Easy to understand

## 0.2 Structure Alignment (0-5)
- Matches story sections
- Proper transitions

## 0.3 Persuasiveness (0-5)
- Problem compelling
- Value clear
- Emotional and/or logical appeal

## 0.4 Redundancy Control (0-5)
- No repetition
- No filler language

## 0.5 Multimodal Sync (0-5)
- Matches visuals
- Matches on-screen text
