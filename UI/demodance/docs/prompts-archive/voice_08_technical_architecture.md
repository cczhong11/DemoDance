# VOICE Section Rubric: Technical Architecture (Optional)
Target duration and words:
- 30 seconds
- 70-80 words

Must include:
1. System overview
2. Key components
3. Data/model flow

Rubric (0-5):
- Word Count: 70-80
- Clarity: understandable by non-technical listeners
- Structure: layered and logical

Penalties:
- Too abstract: -2
- Overloaded with jargon: -1
